# Zen of PHP

## Install

Via Composer

``` bash
$ composer config repositories.mattharley/zen vcs https://github.com/mattharley/zen
$ composer require mattharley/zen:dev-master
```

## Usage

``` php
php > use \MattHarley\Zen\Zen;
php > $zen = new Zen();
php > $zen->importThis();
```

## Testing

``` bash
$ phpunit
```
